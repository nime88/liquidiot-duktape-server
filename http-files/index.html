Liquid-IoT Runtime Environemnt (LIoTRE) API Specifiction

Liquid IoT Runtime Environemnt (LIoTRE) is a Node.js HTTP server which turns an IoT device into an application server so that it can host and run IoT applications. LIoTRE offers a set of RESTFul APIs that can be used to deploy the IoT applications and manage their operations remotely. More information can be found in https://github.com/nime88/liquidiot-duktape-server.
